Bem Vindo ao Sistema Travessa


Aqui é possível cadastrar, assim como visualizar seus Clientes, Funcionários, Livros e Vendas facilmente.

Cadastrar

No fim de cada página, haverá um botão escrito . Nele você abrirá o Formulário de cadastro da respectiva página.
Agora basta preencher todos os campos do formulário e clicar .
Alguns itens terão os símbolos control_point e remove_circle_outline , eles são capazes de cadastrar mais de uma característica específica, como por exemplo, 2 gêneros literários.

Itens Cadastrados

Cada item cadastradado será exibido em uma Caixa, onde todas as suas informações poderão ser encontradas. Note que algumas páginas é necessário usar o botao para realizar o cadastro. Exemplo:

Livro 8

Título: It a coisa
Autor: Stephen King
Gênero: Drama,Suspense
Preço: R$64.99

Procurar Itens

As páginas possuem um campo no topo para pesquisar um item específico. Ele se baseia no nome (título) para encontrá-lo. Caso deseje que a lista toda apareça novemente, é só deixá-lo em branco.

Nota:A procura de vendas se baseia no nome do funcionário que a realizou.
