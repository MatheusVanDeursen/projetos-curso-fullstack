# 📚 Arquivo Histórico: Curso Full Stack 

Bem-vindo ao meu repositório de arquivo legado! 

Este espaço é dedicado a preservar os projetos finais desenvolvidos durante meu primeiro curso Full Stack. O objetivo de manter esses códigos públicos não é exibir as melhores práticas ou arquiteturas modernas, mas sim documentar **o início da minha jornada na programação**. Mais do que exibir código, o objetivo aqui é mostrar consistência, transparência e a evolução técnica contínua, evidenciando a base prática que fundamenta os desafios mais complexos que resolvo hoje na Ciência da Computação.

Por uma questão de transparência e registro histórico, optei por **não refatorar** os arquivos. Eles não utilizam os padrões de código, Clean Code ou a estruturação de arquitetura que domino em meus projetos atuais, servindo puramente como uma "cápsula do tempo" do meu aprendizado inicial. Alguns dos projetos e seus respectivos vídeos de apresentação foram perdidos, mas a maioria deles foi recuperada.

## 🗂️ Estrutura do Repositório

Os projetos estão organizados em pastas, refletindo a minha evolução ao longo dos módulos do curso. Em cada pasta, você encontrará o código-fonte original e um README detalhado (alguns incluindo vídeos de demonstração rodando a aplicação).

* 🌐 **[01. Sistema de Gestão de Livraria (JS SPA)](./modulo-1-html-css-js):** Minha primeira Single Page Application manipulando o DOM nativamente com HTML, CSS e JavaScript Vanilla.
* 💾 **[02. Sistema de Gestão Escolar (Banco de Dados)](./modulo-2-banco-de-dados):** Modelagem relacional em MySQL com foco em chaves estrangeiras, consultas complexas (INNER JOIN) e Stored Procedures.
* ☕ **[03. Locadora de Veículos (Java MVC)](./modulo-2-java):** Construção de uma aplicação Desktop utilizando Java Swing (WindowBuilder) integrada a um banco de dados MySQL via JDBC.
* 🐘 **[04. E-commerce / Loja Virtual (PostgreSQL)](./modulo-3-postgresql):** Evolução em banco de dados, criando a infraestrutura de um e-commerce com uso avançado de PL/pgSQL, Functions e Triggers para regras de negócio.
* 🎨 **[05. Landing Page CompMastery (SASS & Bootstrap)](./modulo-4-html-css-intermediario):** Desenvolvimento Front-End responsivo explorando a componentização do Bootstrap 5 e a organização modular de estilos com SASS.
* 🏦 **[06. Sistema Bancário (JS Orientado a Objetos)](./modulo-4-js-intermediario):** Simulador de console focado em aprofundamento lógico, aplicando herança de Classes, encapsulamento e ES6 Modules.

## ⚠️ Nota Importante sobre o Código

Todo o código-fonte presente neste repositório é estritamente **legado**. Ele reflete o meu nível de conhecimento na época da entrega dos projetos.
